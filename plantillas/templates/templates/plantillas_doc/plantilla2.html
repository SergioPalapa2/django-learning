Shortcuts

La libreria shortcuts recolecta funciones de ayuda y clases que se implementan en distintos niveles de nuestra aplicacion MVC

Fuente: Django shortcuts

  1. Importamos la libreria en nuestro archivo views: from django.shortcuts import render
  2. Utilizamos la siguiente linea al final de nuestra funcion: return render(request,"plantilla2.html")

Plantillas incrustadas

Documentos html que se utilizan dentro de otro documento html.
En Django se pueden visualizar como una plantilla principal que contiene otra plantilla.
Pasos para incrustar plantillas:
  1. Crear una nueva plantilla
  2. Determinar el lugar en donde va a aparecer la plantilla dentro de la plantilla principal
  3. Se incrusta plantilla mediante: {% include "plantilla.html" %}

Es habitual crear dentro de nuestra carpeta plantillas subcarpetas que organicen las plantillas.

Nota: Es importante incluir la ruta de la subcarpeta en el include

Herencia de plantillas

Consiste en crar una plantilla padre sobre la cual se van a basar el resto de plantillas en el sitio web.
Se crea una estructura general que se mantendra en todo el sitio web y tendra uno o varios bloques cambiantes.
Para heredar se utiliza: {% extends "padre.html" %}
Las etiquetas se deben colocar siempre al inicio de las plantillas hijas.
Pasos para heredar de plantillas:

  1. Se crea la plantilla padre en carpeta de plantillas
  2. Se colocan los bloques cambiantes
    • {% block title %} {% endblock %}
    • {% block content %} {% endblock %}
  3. Se crean las plantillas hijas utilizando {% extends "padre.html" %}
  4. En la plantilla hija se indican los bloques que van a cambiar mediante los bloques {}
  5. Crear una nueva vista en archivo views.py, apuntando a plantilla hija (render(request,"hija.html",... )
  6. Se registra la URL
